## Color Palette Reference

**ARIEUS Colors:**
- **Twilight Purple:** #9B59B6 (primary brand)
- **Blood Orange:** #E74C3C (warnings, calls to action)
- **Pastel Horror:** #F8BBD0 (success states)
- **Neon Cyan:** #00FFFF (highlights, tech elements)
- **Dusk Gold:** #FFD700 (premium/value indicators)
- **Void Black:** #1C1C1C (backgrounds, depth)

**Use 2-3 colors max per visual** to avoid overwhelming the design.

## Typography Guidelines

**Recommended Free Fonts:**
- **Headlines:** Orbitron Bold, Rajdhani Bold, Space Grotesk
- **Body Text:** Inter, Roboto, Open Sans
- **Accent/Code:** Fira Code, JetBrains Mono, Courier Prime

**Hierarchy:**
- Profile name: All caps, bold
- Headlines: 2x body text size
- Body: 16-18px equivalent
- Captions: 12-14px equivalent
